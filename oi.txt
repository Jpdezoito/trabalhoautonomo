## Prompt 1 — Prompt mestre do projeto

Cole esse primeiro:

```text
You are a senior full-stack engineer specialized in building modern, scalable, production-ready marketplace platforms with excellent UI/UX.

I want you to build a complete professional service marketplace website inside this project.

The platform is for autonomous workers and service providers, such as:
- mechanics
- builders
- electricians
- plumbers
- painters
- furniture assemblers
- renovators
- appliance repair workers
- technicians
- local service professionals in general

Main goal:
Create a complete, organized, modern platform where:
- workers can create detailed public profiles
- clients can register, search, contact, request quotes, favorite workers, and leave reviews
- administrators can manage the whole platform

The project must be:
- complete
- modern
- responsive
- very well organized
- production-style
- scalable
- visually polished
- modular
- cleanly coded

Required features:
1. Public homepage
2. Search by service type
3. Search by city and neighborhood
4. Worker public profile pages
5. Registration and login
6. Separate flows for client and worker accounts
7. Worker dashboard
8. Client dashboard
9. Admin dashboard
10. Portfolio/gallery system
11. Quote request system
12. Reviews and ratings
13. Favorites system
14. Category management
15. Contact options including WhatsApp
16. Profile verification and moderation
17. Platform settings and configuration areas
18. Professional menus and navigation
19. Mobile and desktop responsive layouts
20. Premium modern design

Recommended stack:
- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- NextAuth

Important rules:
- Build this as a real professional platform, not a toy project.
- Keep everything organized inside the project.
- Do not place everything in a single file.
- Use reusable components.
- Use scalable folder structure.
- User-facing texts must be in Brazilian Portuguese.
- Technical code can be in English.
- Always explain what files you will create or modify before generating code.
- Always preserve architecture quality.
- Build step by step, not in one messy giant block.

Before writing code:
1. inspect the current project
2. propose the ideal architecture
3. define routes, modules, components, layouts, and database models
4. define the implementation roadmap
5. then begin implementation step by step
```
